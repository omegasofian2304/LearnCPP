# Learning C++

This repository contains my C++ learning journey, starting from the basics and progressively moving toward data structures and algorithms.

## Why C++

C++ is the language of choice in quantitative finance and high-frequency trading due to its performance and low-level memory control. This repo is part of my preparation for EPFL CS and a future career in quant development or ML/AI engineering.

## Topics covered

- Variables, types and memory (stack vs heap)
- Functions, pass by value, reference and pointer
- Conditionals and loops
- Vectors and arrays
- Pointers and dynamic memory allocation (new/delete)
- Linked lists
- Binary search trees (in progress)

## Structure

Each concept is implemented as a standalone .cpp file with practical exercises.

## Goals

- Build solid C++ fundamentals before entering EPFL
- Understand low-level memory management
- Implement classic data structures and algorithms from scratch
